# Clash Royale Deck Builder

Find the best Clash Royale decks for every arena.

ðŸ”— **Live Site:** [www.crdeckbuilder.top](https://www.crdeckbuilder.top)

## Features

- Top-performing decks for all 20 arenas
- Win rates and usage statistics for each deck
- Card-specific deck recommendations (Hog Rider, P.E.K.K.A, Giant, Balloon, etc.)
- 104 statically generated pages for fast loading
- SEO optimized with JSON-LD structured data, sitemap, and breadcrumb navigation

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Data:** Clash Royale API

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.
